# Medconnect-AI

MedConnect AI

AI First CRM for Healthcare Professionals



Overview

MedConnect AI is an AI-powered Customer Relationship Management (CRM) system designed for Medical Representatives.

The application enables healthcare representatives to record doctor interactions, analyze engagement, maintain visit history, and interact with an AI assistant powered by LangGraph and Groq.
